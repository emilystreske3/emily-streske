This repository contains Emily Streske's professional brand website built for GitHub Pages.

## Brand Focus

- Chief Operating Officer, Shield Technologies
- Cybersecurity leadership and MSSP operations
- Software engineering and DevOps execution

- Building Titan as a cybersecurity/compliance platform

## ENG 202D Requirement Coverage

- Professional profile website: complete across homepage, about, resume, projects, and insights pages
- Appropriate headshot: included in homepage and about page hero sections
- Headline and summary: included on homepage and profile pages
- Professional social links: email, LinkedIn, GitHub, and company site
- Three blog posts with 200-400 words each:
	- blog/eng202d-cybersecurity-leadership.html
	- blog/eng202d-devops-security-operations.html
	- blog/eng202d-compliance-risk-execution.html
- At least two headings per blog post: included in each featured post
- Credible source link per blog post: included in each featured post
- Relevant image with caption/source per blog post: included in each featured post
- Conversational but professional tone: applied throughout
- Clear navigation and mobile-friendly design: implemented via shared navigation and responsive CSS
